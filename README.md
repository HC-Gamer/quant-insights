# quant-insights — 技术分析教学站 / Technical Analysis Insights (Educational)

A static, public, **educational** site showing daily technical-analysis reads,
backtests, and methodology from a cautious, long-only, paper-first swing-trading
project. Published via GitHub Pages.

**Live site:** https://hc-gamer.github.io/quant-insights/

## 关于内容 / About
- 只展示**描述性**技术面（趋势、RSI、MACD、回调形态）与回测/调参方法论。
- **不含可执行的买卖点位**（入场价/止损/止盈/仓位）——这些刻意从公开版剔除。
- 页面由上游项目自动生成；数据源 yfinance，账户为模拟（paper）。
